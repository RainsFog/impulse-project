# Импульс тестовое задание — BTS Artifact Generator

Генератор артефактов для информационной модели базовой станции (BTS).

## Описание

Программа читает UML-модель базовой станции в формате XML и конфигурационные
файлы, после чего автоматически генерирует 4 артефакта:

- `config.xml` — внутренняя конфигурация станции
- `meta.json` — мета-информация классов для UI
- `delta.json` — разница между старым и новым конфигом
- `res_patched_config.json` — результат применения дельты


## Запуск

```bash
python generate.py
```

## Тесты

```bash
python -m pytest tests/ -v
```

Все 54 теста проходят успешно.
